## API

### 属性

|   参数   |       说明       |  类型   | 默认值 |
| :------: | :--------------: | :-----: | :----: |
|   type   |   设置按钮类型   | string  |   -    |
| disabled | 设置按钮失效状态 | boolean |   -    |
|   size   |   设置按钮大小   | string  |   -    |
|  shape   |   设置按钮形状   | string  |   -    |
|  color   |   设置按钮颜色   | string  |   -    |
