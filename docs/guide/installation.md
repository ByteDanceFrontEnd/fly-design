# 安装

## 环境支持

由于 Vue 3 不再支持 IE11，Fly Design 也不再支持 IE11 浏览器。

## 版本

fly-design 0.3.0

## 使用包管理器

建议您使用包管理器 (npm, yarn, pnpm) 安装 Fly-Design, 然后您就可以使用打包工具，例如 Vite 和 Webpack

# 选择一个你喜欢的包管理器

## NPM

```shell
npm install fly-design
```

## YARN

```shell
yarn add fly-design
```

## PNPM

```shell
pnpm install fly-design
```
